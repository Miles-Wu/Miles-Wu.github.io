考证经验谈

2019-12-28 2019 words 5 mins read times read

Contents

    • 三、备考的过程
    • 四、小工具

今年参加了3次注册考试，写一点心得吧。 准备考试的过程是十分痛苦的，考完之后立刻会忘记那段难熬的时间，以至于现在回望那段准备的时间都已经觉得很遥远。 对备考过程中痛苦的体验，和记得一些知识都开始迅速的忘记。 我今年的准备过程有没有价值呢？我相信多少还是有一点的，朋友同事们都来问我，是如何准备的。没考过的同事想找我分析原因。 虽然找我的人不多。 当人们在一段充满未知的旅程中，总会有一种希望有人能够帮助的心理，一名同伴或者一名指路人，那样会觉得很安心。 今年参加的考试是注册咨询，注册城市规划，一级建造 建筑。文章讨论的内容，就在其中。个人的经验也局限于2018 2019的两次考试中。

一、运气很重要

把注册考试仅仅当成一次考试，仅仅从应试的角度来看，仅以达到60分来看，运气太重要了。这是我后来才认识到的一点。考试书籍那么厚，知识点那么多，但落在卷子上的不过是十分之一，甚至都不到。辛苦背诵了几十个知识点，付出了不少，但有可能是一个点都没考，能气死人。考一级建造师的实务和今年的相关知识就是这样。准备100个点，结果只考10个点。因此，从考试的意义上讲，在临考前去看看各个培训机构的押题班是很必要的，看看他们给出的押题方向中有没有出乎意料的地方

出题人也会去了解培训机构的讲课的侧重点，采用反套路的出题方式，培训班老师重点讲的内容一个不出，那些没有重点讲的地方确实在反复出题。如果有余力，要全面一点的去看书，多了解一些。

不过，考试题目中总会有简单的和基础的题目，这些各个培训机构的都会讲到，这些是必须掌握的，且是必须拿到的分。

备考的过程不要太过去强求自己，也不用太过于扣细节。

二、有多少可备考的时间

每个人的时间不一样，有的人工作清闲，可以有较多的时间看。有的人工作忙，只能花少点时间看，不同的时间有不同的复习策略。对于时间充裕的人来说，这样复习，能够稳妥一些。 证书都是在职考的，时间安排很重要。基本工作日每天要有2个小时的学习时间。休息日得有4个小时的时间。如果能有更多时间，那更好。

如果一直加班，无法满足按部就班式的复习，就得靠一点运气了，靠真题和来。

复习上的时间安排，不宜过早，也不宜过晚。每一门科目，除了极难的科目（2019一建的实务）之外，一般都需要一个月的时间，实务可能会多一周，才能完成精讲。

三、备考的过程

个人的备考思路就是，找到培训班视频，跟着视频走。本人看不进去书，大量的文字极枯燥，能看得进去的就是视频里优化的讲义和重点。按照这个路子，最怕选错了培训机构和讲师，培训市场上有的老师确实很差，如果发现听不进去，要果断抛弃，否则听到后面，和其他人的思路不一样，也不太方便转。

（1）选对讲师

挑对视频很重要，找到能够听得进去的讲课风格很重要。要多问问，多看看，找对以后，就跟着一个培训班走下去就可以。押题的时候再看看其他的培训机构的观点。

（2）看网课视频

在看视频的过程中，如果有讲义就不说，但如果没讲义，要通过电脑截图或者拍照的方式，把视频里的内容截取下来，可以带有一些笔记。以后复习的时候，就可以通过看讲义，迅速回忆起每一章节的。

挑对视频很重要，找到能够听得进去的讲课风格很重要。要多问问，多看看，找对以后，就跟着一个培训班走下去就可以。押题的时候再看看其他的培训机构的观点。

以个人的经验来看，各种视频基本可以快进播放的，正常速度播放可能有点慢，免不了不耐烦和丢失注意力。

看视频过程中，也可以自己在书上画重点。要避免被动的在听，带有目的性的去听很重要。

（3）做真题

考证不需要做其他模拟题，只需把真题做好就可。复习完一门科目后，即可开始做历年真题。

（4）复习和总结

在看一门知识中，要定期去看看当时的讲义截图，留有一定的印象。对于容易混淆的概念要自己想下如何辨析。比较好的状态是，在考前可以在头脑中回忆串联起整个书本的结构和重要的知识点。

（5）考前

考前如果有条件，最好请几天假，把复习过的知识过一遍。包括：

  1. 教材上的重点
  2. 历年真题的错题
  3. 讲义重点
  4. 押题重点

复习完这些以后，就可以安心上考场，做到上述这些已经是尽自己所能了，能不能过就看运气了。

四、小工具

对于准备过程中，有一些迷信。要有比较好的工具。

  1. 要备好黑色和红色的中性笔，一到两种的记号笔。把重要的东西醒目的标出来，能提高不少效率。
  2. 铅笔、橡皮和直尺，计算器。
  3. 文件档案袋。
  4. 手表。

最后

考证的过程是阉割人性的过程。极其枯燥和难熬，要放弃不少的社交活动。忙往工作后还要集中注意力看书，身体要得要抗的住。对于考证的意义，个人认为其实不是特别大，如果在手的项目质量好，大可不必花太多的时间在证书上，不会因为有证书就可以增加一些能力。但如果有时间，又无其他事可做，可以认真学一学，考一下，毕竟还有可以学到一点知识，备考的过程能是一个不断调整的过程，对于其他事还是有些帮助的。
