自己在2019年初经过三个月的复习，侥幸通过了注册咨询工程师的四门考试，并在10月份拿到证书，完成了注册。临近2020年，又到了咨询工程师报名和考试的时候，周边的一些朋友开始准备新一年的考试。对于咨询工程师这项考试，自己有一点认识，在此写篇文章，希望能够对未来参加这项考试的朋友有所帮助。

参加一项考试要大概要经过这么几个阶段：认识考试、报名、备考复习和注册登记，按照这个顺序逐一介绍。

一、认识考试

这个证有什么用？

大部分人对注册咨询工程师是比较陌生的，不知道这个执业资格主要从事的是什么工作，以及这个证有什么用。其实，咨询工程师的主要工作之一是完成可行性研究和项目建议书这类文件，在工程建设早期服务政府做固定资产投资决策。可行性研究和项目建议书这类文件要解决的重要问题之一是评估一项固定资产投资是否值得。比如投资数十亿建一条地铁是否值得？在镇上新建一座污水厂是否值得？投资几千万建造一处回迁房是否值得？这是咨询工程师要主要解决的问题，也是咨询工程实务考试里的难点。这项注册制度的目的就是对一个从事工程咨询行业从业人员的执业水平进行认定。

而且，现在政府要求一些可行性研究文件必须要由注册咨询工程师的盖章认定。对于一些在项目前期就能够与政府或企业对接的设计院，一般是综合型国企设计院或专业的咨询公司，就需要有一批注册咨询工程师。一定数量的注册咨询工程师也是申请和提升工程咨询资质的必要条件，更高的咨询资质能够提升企业的竞争力，在目前这个经济形势下还是挺重要的。

这个证书对个人来说还有一个小用处，就是可以用于个人所得税专项附加扣除（虽然抵扣不了多少）。

证书的来源

工程咨询行业最顶层的管理者是国家发改委，具体的指导和监管机构是中国工程咨询协会。国家发改委的重要职责之一就是规划重大建设项目，也就是对大型工程建设项目进行决策。在地方各级政府层面上，国家发改委希望有高质量的工程咨询企业来辅助各级地方政府做出科学合理的投资决策和进行建设过程管理，因此，发改委制定了《咨询工程行业管理办法》对这一行业进行约束，管理的方法之一就是设立了注册咨询工程师的制度。

证书的特点

这个证书有一个重要特点是，在注册时提供了十几种可选专业，覆盖面很广，且考前不需要选专业，不同专业的考试内容都一样。初始登记时可同时申请注册两个专业，中国工程咨询协会根据申请者所学专业和所在公司主要从事的项目类型进行审批。

可以注册的专业有以下这些，来自《咨询工程行业管理办法》：

（一）农业、林业；（二）水利水电；（三）电力（含火电、水电、核电、新能源）；（四）煤炭；（五）石油天然气；（六）公路；（七）铁路、城市轨道交通；（八）民航；（九）水运（含港口河海工程）；（十）电子、信息工程（含通信、广电、信息化）；（十一）冶金（含钢铁、有色）；（十二）石化、化工、医药；（十三）核工业；（十四）机械（含智能制造）；（十五）轻工、纺织；（十六）建材；（十七）建筑；（十八）市政公用工程；（十九）生态建设和环境工程；（二十）水文地质、工程测量、岩土工程；（二十一）其他（以实际专业为准）。

注册咨询工程师的另一个特点是，它属于水平评价类职业资格，与之相对应的是准入类职业资格，比如一级建造师、注册城乡规划师和一级注册建筑师等等，相比之下注册咨询工程师的考试难度和含金量要比这些准入类的执业资格要低一些。

要考哪些内容？

一共四门考试：

  1. 《项目决策分析与评价》（后称为《决策》）全为选择题。主要讲的是可研和项目建议书等报告的内容、方法和适用条件。难度一般，学习此科目可以对工程咨询有个大致了解，个人感觉这一科是备考这项考试较好的入手点。19年的考试，把《实务》里的部分计算题放在《决策》这一科里考，提升了该科的难度。

  2. 《宏观经济政策与发展规划》（后称为《宏观》）全为选择题。讲的是一些宏观经济原理和实事政策。难度较小，教材内容可与时事关联，比较容易接受。

  3. 《工程项目组织和管理》（后称为《管理》）全为选择题。涉及面广，内容繁杂，是一个知识大杂烩，与一级建造师的管理和经济科目的部分知识有重叠。个人感觉这一科是除实务以外难度最大的一科。该科目复习时一定要选好讲课老师，水平一般的老师会不分主次，区分不出重点，降低学习效率。

  4. 《现代咨询方法与实务》（后称为《实务》）全为主观题，包括简答和计算。包含财务分析和经济分析的内容，难度较大，涉及大量的计算。对具体的咨询实践有一定的指导意义。

四科满分均为130分，及格线均为78分。

哪些人可以考此证？

  1. 工程咨询行业的从业者，或想进入此行业的人。

  2. 可以靠这项证书带来一定收入的人。

考试难度如何？

对比参加的一级建造师和注册城乡规划师来说，咨询算是比较简单的考试，建议一次报4科，花些时间学的话，可以一次通过。

二、报名考试

报名时间和考试时间

报名时间是1月下旬，一般在农历新年前后。考试时间一般在4月中旬（目测2020年是4月11日和4月12日）。

报考条件

自行百度之。

报考费用

咨询工程师是比较贵的一门考试，考试费用是500多元。四套正版教材是300多元，还没开始学就要花掉800，而且教材是每年更新（协会真会做生意）。如果再买一些视频课，就需要近1000元的费用，算是在各类考试里比较贵的。

三、备考复习

需要学习多久？

3个月足矣，平均每天3小时左右。

实务的精讲阶段差不多需要1个月的时间，其余三门可以在3周内搞定。留下2-3周做最后的复习。

四科的顺序是什么？

自己学习的顺序是三门公共课看完以后再看实务。但考完以后的感觉是，实务可以提前看，因为其内容较多，如果放到最后看，可能因为时间紧张而复习不完。《决策》可以先看，作为对咨询工程的了解。因此，推荐的学习顺序是《决策》—《实务》—《管理》—《宏观》。

学习资源在哪找？

淘宝有不少便宜的教学视频资源，需要自己搜索。

学习方法？

参见我的一些考证经验。

考试成绩公布时间？

考试成绩一般会在6月中旬公布，是一年中所有注册考试中公布成绩最早的。

五、注册登记

如果通过了考试，需要在中国工程咨询协会网站上进行登记。登记时需要提交的各类资料在网站里都有说明。需要注意的是，初始登记需要在当前季度的第2个月的20号前提交申请，否则就只能等到下个季度才能进行审核，相当于要多等一个季度。对登记有时间要求的人，需要尽快登记。

目前的证书和注册章基本都已经数字化了，比较坑的是如果想要盖电子章，必须购买指定厂家的U盘，还要再向协会的合作单位付一笔钱。

以上就是对咨询师备考的大致介绍。要成为一名注册咨询师不容易，但值得为此付出努力，希望在2020年参加考试的朋友都能取得好成绩~
