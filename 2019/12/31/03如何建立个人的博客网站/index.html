电脑小白如何搭建个人的博客网站

在朋友jdhao的帮助下，终于建成了自己的博客。尽管朋友写了一篇教学文章，但作为电脑小白照着操作还是有点困难，最终是在朋友的远程帮助下，建成了这个博客。 朋友的文章有一定的门槛，不是针对小白写的。自己在搞清楚这个过程后，从0基础的角度来整理下整个建站的过程。但如果要学习这个过程，依然要参考这篇教学文章。

基本原理

对搭建个人网站原理的粗浅理解是，首先在本地建立一个文件夹，里面存放文章，然后在网上建立一个远程托管仓库（可被其他人访问），然后将二者关联起来。因此搭建个人网站大概分为以下三步：

  1. 建立一个本地博客站点
  2. 申请一个远程托管仓库
  3. 将本地站点部署在远程仓库上

本人使用的是windows系统，以下均以windows环境为例来说明整个过程。

（一）建立本地博客站点

step1 下载并安装hugo

Hugo是一个建立网站的工具，用hugo可以生成一个本地的仓库。但在windows上安装hugo之前需要安装一个包管理工具叫做chocolatey。

在安装chocolatey时，根据官网的说明，是需要进入到powershell中。

在左面左下角处右键点击“搜索”，输入powershell，右键以管理员身份运行，即可打开。然后参照官网的说明操作：

在命令行里输入Get-ExcecutionPolicy，然后回车。

如果返回的是Restricted，接下来输入Set-ExecutionPolicy AllSigned or Set-ExecutionPolicy Bypass -Scope Process.

完成后，复制官网上给出的下载命令，并运行，即可下载安装好chocolatey。

安装好chocolatey后，在powershell中运行命令choco install hugo -confirm，等待片刻即可安装好hugo。

结束后不要关闭powershell。

step2 新建博客文件夹

首先要确定把本地的博客文件夹放在哪个盘下，如果想要放到D盘，首先要将命令行的开头地址改为D盘，方法是在powershell中输入命令cd .. ，此命令是返回到上级菜单。如果默认命令行是C盘里的一个文件夹，则需要运行该命令数次，知道返回到C:\。

然后输入cd D:\，回车后可以看到命令行的开头变为D盘，接下来就可以在D盘目录下建立文件夹。

建立的方法是在powershell中运行hugo new site Blog命令。

然后就可在D盘中看到一个名称为Blog的文件夹，里面还有数个文件夹，其中包含这几个：

  1. config.toml 为配置文件
  2. content/ 为存放文章的文件夹
  3. themes/ 为存放主题的文件夹

如果想修改文件夹存放的地址或者修改名字，直接在文件资源管理器中将其删除，然后用上述方法重新建立即可。

step3 安装git

git是一种分布式版本控制系统，在官网上即可下载git，安装过程并不复杂，直接下一步就好。

step4 安装visual studio code

visual studio code是一个文本编辑器，在官网上下载，安装也很简单。这个工具用来编辑config.toml，写博文也是要用到这个编辑器来书写。

step5 安装并设置博客页面的主题风格

在powershell中用cd命令将地址改到blog这个文件夹中。逐行运行下列命令：

git init

git submodule add https://github.com/olOwOlo/hugo-theme-even.git themes/even

等待片刻，即可下载好目前本博客使用的even主题。Hugo也提供了一个主题网站，自己可以根据喜好来设置。

用visual studio code打开config.toml，将theme命令行改为下列内容：

theme = "even"

至此，配置主题成功。

step6 新建一篇文章

在powershell或者cmd中的D:\blog>运行下列命令：

hugo new post/test.md

test是这篇文章的标题，可以更换，用汉字输入亦可。

这个命令会在content文件夹中，生成一个post的文件夹，并在其中生成test.md，博客的书写就是用visual studio code书写，并需要遵循markdown的语法。找到test.md，并用visual studio code打开后，先在其中输入一些内容，输入完成后，要将draft选项改为false。改为这样，才能够在网站上发表。

step7 在本地服务器上预览博客

在powershell或者cmd中运行下列命令：

hugo server

会看到一些结果，看不懂没关系，然后在localhost：1313中即可预览到博客的效果。

至此就完成了在本地建立博客站点，下一步就是要申请一个远程仓库。

（二）申请一个远程托管仓库

一般使用免费的github来建立仓库。

step1 注册github账号并登陆

这个没什么说的，github非常友好，很轻松的就可以注册上。

不过此处注意，github的username就是未来个人网站的域名，建议取一个简洁、有含义、便于辨识和记忆的名字。这是一个听花费时间的步骤，因为会发现好多名字被占用了……

step2 在github上建立一个仓库

点击右上角的repositories，然后再点击new，即可建立仓库，仓库的名称需要注意，应写为：

username.github.io

这就是未来个人博客的域名了。

用户名的修改可以在settings中account中设置。

（三）将本地站点部署在远程仓库上

取了一个满意的名字后，就可以开始将本地的文章部署在github上了。

step1 在blog文件夹内建立public文件夹

在D:\blog>里逐行运行下列代码:

mkdir public`

cd public`

step2 关联github的仓库地址

在D:\blog\public>里逐行运行下列代码:

git init

git remote add upstream https://github.com/ycwufan/username.github.io.git

step3 将blog的内容放到public

此步骤需要在每次发表新文章时重复运行，可通过脚本替代。

在D:\blog>里逐行运行下列代码:

hugo -t even

step4 将blog的内容push到github上

此步骤需要在每次发表新文章时重复运行，可通过脚本替代。

在D:\blog\public>里逐行运行下列代码:

git add .

git commit -m  "switch to hugo"

git push -f upstream master # 第一次 push 必须使用 -f，因为此时远程仓库非空，和本地有冲突

至此，博客就可成功发表，等待几分钟，在浏览器中输入域名，就可以看到自己的文章。
